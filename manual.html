AI 聊天记忆管家

明月秋青智脑

你在酒馆和角色聊天，它在后台默默帮你 记、理、用。 所有的角色记忆、剧情时间线、关系变化，它都打理好了，你只管聊。

记

每轮对话后自动总结刚才发生了什么、出现了什么新物品和新地点。聊久了还会做一次"大总结"，把整段剧情整理成清晰的时间线，写入每个角色的记忆档案。

理

分析角色之间的关系变化、每个角色的性格随着剧情怎么演变、你的互动风格有什么特点。还会把故事里出现过的地点和物品串成一个知识网。

用

每次 AI 要回复你的时候，智脑从记忆库里挑出最相关的内容，悄悄塞进上下文。所以角色聊着聊着就会"记得"你们之前一起经历过的事。

安装

导入一行脚本，30 秒搞定。

1 打开酒馆 → 顶部菜单"扩展管理" → "酒馆助手" → "脚本" → "新建脚本" → "预设脚本"
2 在输入框中粘贴下方地址
3 点击确认。稍等几秒，出现紫色的悬浮窗即安装成功
→ import 'https://cdn.jsdelivr.net/gh/sillytavner-jpg/zhino-script@v4.7.2/dist/index.js'
更新版本：扩展管理 → 酒馆助手 → 脚本，找到智脑脚本，点编辑，把地址里的 @v4.7.2 换成最新版本号即可。

五分钟配置

装好之后做三件事：配 API、开向量召回、填一份你的人设。之后就只管聊。

第一步：配置 API

打开智脑面板 → 设置 → 网络与 API → 管理 API，在这里添加你的 API。智脑的后台任务需要调用模型来生成总结和记忆，推荐按以下分配：

任务类型推荐模型说明
大总结 / 角色记忆 / 世界推进Gemini 3.1 Pro 或 DeepSeek V4 Pro需要强理解力，不要用小模型
小总结Gemini 3 Flash 或 DeepSeek V4 Fast轻量任务，快速模型即可
其余全部Gemini 3.1 Pro动态人设、梦呓、关系分析等
调度模式：在管理 API 页面的"调度模式"中，如果你的渠道支持同时 5 个并发调用，或者你给不同功能分别配了不同渠道的 API，建议开启并发模式——多个任务同时跑，体验更流畅。否则选排队模式，任务按顺序逐个执行。

第二步：开启语义召回

设置 → 高级功能 → 召回设置 → 开启"语义召回"。推荐使用硅基流动（SiliconFlow）的免费向量模型 BAAI/bge-m3：

  • 去 siliconflow.cn 注册，在 API 密钥页面复制你的 Key
  • 回到智脑召回设置，粘贴 Key，模型填 BAAI/bge-m3
  • 地址默认即可（https://api.siliconflow.cn/v1）
  • 填好后点一下测试连接，显示成功就配置完了

语义召回能让记忆匹配从"关键词"升级为"含义理解"——比如能认出"打架"和"起冲突"是同一件事。长篇多角色时效果显著，短篇可以先不开。

第三步：填写你的用户人设

1 打开智脑面板，点击 玩家 标签，进入 用户人设
2 点击 新建，用几句话描述你自己——性格、说话风格、习惯都行，不用写很长
3 点击 保存并分析，AI 自动生成你的互动画像。分析完成后角色就能更懂你的风格
完成后：先正常聊几十轮，再打开智脑面板看看。角色记忆、动态人设、剧情时间线都已经自动生成了。之后只需根据自己的习惯微调设置即可。

面板指南

面板共 6 个顶层标签，大部分内含二级子标签。以下逐个展开。

总览

仪表盘首页，运行状态一览 + 手动操作入口。

统计面板 — 当前楼层、距下次大总结轮数、累计总结次数、已捕获回复数、世界推进/剧情导演开关状态、动态人设版本、物品库条目数。

活跃角色 & 梦呓状态 — 当前故事中出场的角色列表，你的互动画像是否已生成。

最新大总结 — 版本号、生成时间、覆盖楼层范围、涉及角色数、事件数。

手动总结 — 立刻触发一次大总结。适合剧情告一段落时用。

重新总结 — 上次总结失败或写偏了，用同样内容重试。

自选楼层总结 — 只总结你指定的某一段聊天。

批量总结 — 导入旧聊天记录后，选起始结束楼层和每批大小，一口气消化所有历史对话。

楼层隐藏管理 & 加载历史总结 — 排除特定楼层、查看过往版本。

记事本

5 个子标签，覆盖所有"记录类"功能。

小总结
每轮对话的一句话摘要。状态分待生成/就绪/失败/已吸收/已忽略。可编辑时间、地点、在场角色、事件文本，也可手动补总结。
时光轴
按版本排列事件时间线，支持搜索。每条显示剧情时间、重要度星级、触发关键词。可编辑、删除、手动添加事件。
物品库
故事中所有物品自动收录。名称、别名、描述、归属、当前状态。可新增、编辑、删除。
地点库
所有地点自动收录。关键在关系网络：包含子地点、连通路径、所在物品和角色。可编辑关系和新增。
图谱
地点和物品的 SVG 交互关系图。地点圆形节点 + 角色小圆分布周围，玩家特殊高亮。四种连线：包含（实线蓝）、连通（虚线绿）、角色位置（虚线橙）、玩家位置（虚线紫）。支持缩放拖拽，点击节点看详情，自动聚焦玩家位置。

人物志

角色记忆与关系分析，2 个子标签。

角色库

左侧角色列表 + 态度图标。右侧展示别名、核心记忆（永久）、近期记忆（自动替换）、动态人设（随剧情演变）、激活关键词、NSFW 记忆（折叠）。

追忆：按大总结版本分组展示全部历史记忆，可编辑、升降级（核心 ↔ 近期）。召回设置：每角色单独设召回上限；关掉开关后该角色全部核心记忆持续注入。编辑模式下支持合并角色和忽略。

关系网

三栏布局：选择角色 → SVG 关系图（实线 = 已分析档案，虚线 = 记忆推断）→ 选中关系详情（类型、来源、状态、张力、禁止误读）。不会自动更新，剧情转折时手动点「分析选中角色」。

世界

世界观层面，3 个子标签。都需先在设置页开启。

世界推进

后台推演不在场角色。可设优先推演角色（最多 3 个）和世界书参考。每条记录含各角色的行动卡片（位置、行动、原因、可能相遇）+ 背景事件 + 未解决剧情钩子（黄色标记）。

剧情导演

选篇幅后用对话方式和 AI 共创大纲。激活后每轮自动校对，显示进度条和阶段详情。校对结果分正轨/超前/落后/偏离，偏离时给出引导。支持防剧透模式。

世界书蒸馏

选世界书和条目 → AI 逐字提取原文归类，不改写只归类。结果分三层（世界观/角色/条目），可逐层展开编辑后保存写入世界书。

玩家

关于你的互动画像和人设管理，2 个子标签。

梦呓

AI 分析你的互动模式，产出"行为翻译手册"——当你做 X 时 AI 应理解为 Y、不误解成 Z。内容涵盖：基础信息、性格调色盘、肢体接触习惯、说话方式、情绪表达（每种情绪 → 表现 → 禁止误读）、按角色分组的交互模式、Roll 偏好。所有字段可手动编辑，支持撤回/恢复/重新分析。

用户人设

多人设管理系统。新建、改名、删除、切换激活。编辑区填写你的性格和行为模式，点「保存并分析」AI 自动生成互动画像，「仅保存」跳过分析。分析结果可直接编辑。适合不同世界观切换不同人设。

常见问题

先打开总览，点一下读取历史楼层让智脑知道有哪些对话。然后用批量总结消化——每批 30 层左右比较合适，楼层特别多的话可以设到每批 50 层。总结完成后，去设置 → 高级功能 → 召回设置，点「为旧数据生成向量」，等进度条跑完就行了。小总结不用补。

先检查 API 设置——有可能是 API 配置的问题，确认 Key 和地址都没填错。如果不是 API 的问题，去设置 → 网络与 API，打开API 监听器，再触发一次分析，查看 API 的请求和回复。如果回复内容正常、也没有截断，带上你的截图和 API 回复找常陆茉子反馈。

在原设备上打开智脑面板 → 设置 → 数据管理 → 导出数据，会下载一个 JSON 文件。把文件传到新设备，在新设备上打开同一个聊天，进入智脑设置 → 数据管理 → 导入数据，选择文件即可。注意：导出导入只针对当前聊天，不同聊天之间的数据是独立的。

它会自动重试最多 3 次（每次间隔越来越长）。如果等了一会儿还不动，点中止然后把每批的楼层数量调小一点再试。

放心重 roll，智脑会自动处理。重 roll 某条 AI 回复后，对应楼层的小总结会重新生成并替换旧记录，知识图谱也会同步更新。

打开总览，用撤回功能把大总结撤回到删除楼层之前的版本。图谱也有独立的撤回按钮，同样撤回到对应版本。世界推进里的条目需要手动删除——找到被删除楼层范围内的记录，逐条删掉即可。

打开人物志 → 角色库，点编辑角色，在角色右侧点击合并按钮，选择要合并的目标角色，确认即可。智脑已经内置了别名自动归一化，但偶尔 AI 还是会用完全不同的称呼，手动合并就能解决。

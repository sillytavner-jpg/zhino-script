# 更新说明 A4.9.1

- 适用：明月秋青脚本-秋青 A4.9.1
- 发版时间：2026-07-06

## 本次改动

- 新增「设置 → 网络与 API → 自定义破限词」，可按后台 API 调用类型分别调整破限头部和尾部。
- 补齐所有含「秋青子 / Qiuqingzi」的默认破限片段，并处理 `{{user}}` 到当前用户名的发送前替换。
- 修正自定义破限词覆盖时的旧提示残留：自定义头部后会清理后续旧秋青子提示，自定义尾部可完全替换默认尾部。
- 人设分析接入同一套破限词覆盖逻辑；已废弃的倒果为因不再出现在破限词设置和 API 分配列表。

## 通过 CDN 导入酒馆助手

- 导入文件：把 `mingyue-qiuqing-A4.9.1.json` 在酒馆助手「导入脚本」加载
- 或 URL 导入：`https://cdn.jsdelivr.net/gh/sillytavner-jpg/zhino-script@v4.9.1/mingyue-qiuqing-A4.9.1.json`

> 旧用户需要重新导入本版薄壳，或把原脚本 content 中的版本 tag 改为 `@v4.9.1` 后刷新。
